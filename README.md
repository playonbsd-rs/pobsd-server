### POBSD-SERVER

This is the official repository of the 
[pobsd.chocolatines.org](https://pobsd.chocolatines.org)
website that offers a frontend to the 
[PlayOnBSD database](https://github.com/playonbsd/OpenBSD-Games-Database).
